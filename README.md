# Service Booking Web Application

## Project Overview

This project is a simple Service Booking Web Application developed using HTML, CSS, and JavaScript. The application allows users to browse different laundry services, add services to a cart, view the total amount, and book the selected services.

The project was created to practice front-end web development concepts such as responsive design, DOM manipulation, event handling, form validation, and local storage.

## Features

* Navigation bar with Home, Services, Contact Us, and About Us links
* User profile section in the navigation bar
* Dynamic service display
* Add Item functionality
* Skip Item functionality
* Cart management using a table
* Automatic total amount calculation
* Booking form with validation
* Username update after successful booking
* Logout functionality
* Local Storage support for saving username
* Responsive layout for desktop, tablet, and mobile devices
* Font Awesome icons for user and logout actions

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome

## Project Structure

* index.html – Page structure and content
* style.css – Styling and responsive design
* script.js – Application logic and event handling

## Working

1. The application displays one service at a time.
2. Users can add a service to the cart by clicking the Add Item button.
3. The Skip Item button displays the next available service.
4. Added services are shown in the cart table.
5. The total amount is updated automatically whenever a service is added.
6. Users enter their details in the booking form.
7. After clicking Book Now, the booking details are displayed below the button.
8. The entered username is shown in the navigation bar.
9. Logout clears the user information and cart details.
